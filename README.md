#### Description:
Sayings is a full-stack social web application designed to foster a community around impactful quotes and ideas. Built as my CS50x final project, Sayings combines inspiration, learning, and meaningful interaction in a clean and engaging platform.
The application features:
Global feed: Curated quotes from books, movies, and influential figures.
Community contributions: Users can post their own quotes and engage with others.
Reddit-style interaction: Upvotes, downvotes.
User accounts: Secure authentication and personalized experience.
Personalized feed: Algorithms track user engagement to surface relevant content.
#### Tech stack:
Backend: Django & Django REST Framework (DRF) for secure, scalable RESTful APIs.
Frontend: React.js and Tailwind CSS for a responsive, intuitive UI.
Database: PostgreSQL with optimized relational schemas for complex social interactions.

Sayings is designed for people who seek meaningful social interaction, inspiration, and insight. This project reflects my dedication to creating a purposeful, user-centered web experience.
